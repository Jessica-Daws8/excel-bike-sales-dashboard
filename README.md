# excel-bike-sales-dashboard
Excel data analysis project analyzing customer demographics and purchase behavior using pivot tables, charts, and slicers.

Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Interactive Slicers

Dashboard Features
 The dashboard includes:
 - Average Income Per Purchase
 - Customer Age Brackets
 - Customer Commute Distance
 - Interactive Filters for:
    - Marital Status
    - Region
    - Education Level
  
Key Insights
 - Higher income groups were more likely to make purchases.
 - Certain age brackets showed stronger purchasing patterns.
 - Commute distance influenced buying behavior.
